Challenge

Building a full-stack e-commerce with secure, role-based access.

The project required a complete shopping experience — customer-facing storefront, authenticated sessions, admin control, and a clean separation between API and presentation layers.

Approach

API-first architecture with JWT auth and a dedicated admin panel.

Built with Node.js and Express, organized into controllers, services, and repositories. SQLite ensures lightweight portability. Every route is protected by role-aware JWT middleware.
