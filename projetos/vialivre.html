ViaLivre AI

App mobile nativo (Android/iOS) que aprende suas rotas e avisa o melhor horário para saída — com mapa visual, histórico de trajetos e notificações push, construído sobre uma API própria que cruza dados de trânsito em tempo real.

Problema

Saber o melhor horário pra sair exige checar trânsito manualmente todo dia.

A maioria dos apps de mapa mostra o trânsito de agora, mas não aprende seus trajetos recorrentes nem avisa proativamente antes de você precisar sair de casa.

Solução

Um app nativo que conhece suas rotas e avisa antes de você perguntar.

O ViaLivre AI guarda rotas favoritas, cruza dados de trânsito em tempo real via APIs de mapas e dispara uma notificação push com o melhor horário de saída — com mapa visual e histórico de trajetos direto no app.

Tecnologias

Stack

React Native + Expo

App nativo para Android e iOS a partir de uma única base de código, com build e deploy gerenciados via Expo.

Node.js + Prisma

API própria em Node.js com Prisma sobre PostgreSQL, persistindo rotas favoritas, histórico e perfis de usuário.

Dados de trânsito em tempo real

A API própria cruza serviços externos de mapas e trânsito para calcular o impacto real no tempo de trajeto.

Notificações push

Alertas nativos avisando o melhor horário de saída antes de você precisar checar o app.

Mapa e histórico

Visualização de rota em tela e histórico de trajetos salvos, com múltiplas rotas favoritas por usuário.

Login e perfil

Autenticação e perfil de usuário — cada pessoa com suas próprias rotas, histórico e preferências.

Resultado

Da API de horários a um app mobile completo, em build para Android e iOS.

O ViaLivre AI evoluiu de uma API de recomendação de horários para um produto mobile de ponta a ponta — login, mapa, histórico, rotas favoritas e notificações push, tudo sobre uma base full-stack própria. Ainda em build via Expo, a caminho das lojas.
