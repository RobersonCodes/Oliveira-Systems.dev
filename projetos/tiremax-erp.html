TireMax ERP

Sistema de gestão multi-tenant para redes de borracharias — estoque, vendas, automação de atendimento via WhatsApp e um painel SuperAdmin para operar múltiplas lojas a partir de uma única base.

Desafio

Gerenciar estoque e vendas de várias lojas exige mais do que uma planilha por filial.

Redes de borracharias costumam operar com controles manuais ou sistemas isolados por loja, sem visão consolidada nem automação de atendimento. O desafio era construir um ERP que isolasse os dados de cada loja com segurança, mas permitisse uma visão central de operação.

Solução

Multi-tenant real, automação de atendimento e um painel central para administrar tudo.

O TireMax ERP isola dados por tenant a nível de banco, com cada loja operando de forma independente sobre a mesma infraestrutura. A automação de atendimento via WhatsApp (Evolution API, rodando em VPS própria) cobre confirmações e notificações sem intervenção manual, com Resend cuidando do envio de e-mails. Um painel SuperAdmin permite acompanhar e operar todas as lojas a partir de um único lugar.

Tecnologias

Stack

Node.js + Prisma

API em Express com Prisma sobre PostgreSQL, hospedada no Railway, com isolamento de dados por tenant.

React + Vite + Tailwind

Frontend hospedado na Vercel, com interface responsiva para uso no balcão de atendimento.

WhatsApp via Evolution API

Automação de mensagens e confirmações rodando em uma VPS própria na Hostinger, sem depender de serviços de terceiros.

Multi-tenancy

Cada loja opera isolada sobre a mesma base de dados, com regras de acesso que impedem cruzamento de informações entre tenants.

Painel SuperAdmin

Visão consolidada de todas as lojas, permitindo operação e suporte central sem acessar cada tenant individualmente.

Resend

Envio de e-mails transacionais — confirmações, recibos e notificações administrativas.

Resultado

Um ERP multi-tenant em uso real, automatizando atendimento de ponta a ponta.

O TireMax ERP demonstra capacidade de projetar arquitetura multi-tenant de verdade — não só múltiplos clientes na mesma tabela, mas isolamento real de dados, automação de atendimento via WhatsApp e um painel central de operação. É um sistema B2B em produção, resolvendo um problema operacional real de redes de borracharias.
