# Oliveira Systems

Site profissional para marca pessoal/empresa tech.

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Como usar

Abra o arquivo `index.html` no navegador ou use a extensão Live Server no VS Code.

## Personalização

- Troque links de GitHub/LinkedIn se necessário.
- Substitua o texto dos projetos conforme seus repositórios evoluírem.
- Use o logo RO como imagem de perfil e o site como portfólio.
