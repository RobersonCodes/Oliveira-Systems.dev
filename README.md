# Oliveira Systems

🚀 Desenvolvimento de sistemas modernos, APIs REST, backend escalável e inteligência artificial aplicada.

## Sobre

Oliveira Systems é uma marca focada na criação de soluções digitais profissionais, com arquitetura moderna, performance, organização e experiência visual premium.

O projeto reúne aplicações reais desenvolvidas com foco em:

* backend enterprise
* APIs profissionais
* autenticação JWT
* monitoramento de sistemas
* inteligência artificial
* escalabilidade
* arquitetura limpa

---

## Projetos

### Guardian Recovery

Plataforma enterprise de monitoramento, incident response e disaster recovery.

### Sistema de Saúde Municipal

Sistema completo para gestão pública de saúde com APIs robustas e IA aplicada.

### Minha Loja

E-commerce full stack com autenticação JWT, painel administrativo e arquitetura em camadas.

### ViaLivre AI

Solução inteligente para mobilidade urbana com inteligência artificial e análise de rotas.
